# Web Page Summarizer

A Lightweight Python Script Built For Jupyter Notebooks That Scrapes Webpage Content And Produces A Clean Markdown Summary Using A Locally Running Ollama LLM.

## Features

- Fetches And Parses Web Pages With `requests` And `BeautifulSoup`
- Extracts Page Title And Main Text
- Summarizes Content Using An Ollama Model (Default: `mistral`)
- Renders Results Nicely In Jupyter Via Markdown

## Requirements

- Python 3
- `requests`
- `beautifulsoup4`
- `ipython`
- Ollama Running Locally
